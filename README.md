📈 NIFTY50 Stock Market Analysis Using Python
📌 Project Overview

This project focuses on analyzing historical closing prices of NIFTY50 stocks using Python.
The objective of this analysis is to identify stock market trends, volatility, returns, correlations, and investment insights through data analysis and visualization techniques.

The project demonstrates practical implementation of:

Data Cleaning

Exploratory Data Analysis (EDA)

Financial Analysis

Statistical Analysis

Data Visualization


🎯 Objectives

✅ Analyze historical stock closing prices

✅ Identify stock market trends and patterns

✅ Calculate daily and cumulative returns

✅ Measure stock volatility and risk

✅ Perform correlation analysis between stocks

✅ Visualize stock performance using graphs and charts

📊 Analysis Performed


🧹 1. Data Cleaning & Preprocessing
Checked missing values
Removed duplicates
Converted date column into datetime format
Set date as index
Sorted dataset chronologically


📈 2. Trend Analysis


Analyzed stock price movement over time
Compared multiple stock trends
Visualized long-term growth patterns


📉 3. Daily Returns Analysis

Calculated percentage change in stock prices to measure daily performance.

Formula:
DailyReturn=((Pt−Pt−1)/Pt−1)∗100


📊 4. Moving Average Analysis

Applied:

7-Day Moving Average
30-Day Moving Average

Used to identify long-term trends and smooth fluctuations.


🔥 5. Correlation Analysis
Created correlation matrix
Generated heatmap visualization
Identified relationships between stocks


📌 6. Volatility Analysis

Measured stock risk using standard deviation of returns.

Insights:
High volatility = High risk
Low volatility = Stable stock


📅 7. Monthly Performance Analysis
Calculated monthly average prices
Identified monthly market patterns


📈 8. Cumulative Returns Analysis

Analyzed overall investment growth over time.


📉 9. Distribution Analysis
Histogram plots
KDE plots
Return distribution analysis


🚨 10. Outlier Detection

Used Z-Score technique to identify abnormal stock movements.


📌 11. Risk vs Return Analysis

Compared:

Average Returns
Volatility (Risk)

Helped identify balanced investment opportunities.


🏆 12. Top Gainers & Losers

Identified:

Best performing stocks 📈
Worst performing stocks 📉

Based on total percentage returns.
